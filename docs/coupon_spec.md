# クーポン・割引機能 仕様

## 概要

出品者が自分の商品向けにクーポンを発行し、購入者が注文時に適用できる機能。

## 基本ルール

| 項目 | 内容 |
|------|------|
| 発行者 | 商品の出品者（user） |
| 適用対象 | 特定の商品に対して適用 |
| 割引種類 | 固定額（例: 500円引き）/ 割合（例: 10%引き） |
| 割引上限 | 対象商品の金額まで（マイナスにならない） |
| 併用 | 1注文につき1クーポンまで |
| 利用制限 | 有効期限あり / 1ユーザーにつき1回まで |
| コード形式 | 出品者が任意の文字列を設定（システム全体で一意） |
| 適用タイミング | 注文確定時にクーポンコードを入力 |
| キャンセル時 | クーポンを未使用に戻す |

## 適用ロジック

1. 購入者が注文確定時にクーポンコードを入力
2. クーポンの有効性を検証
   - コードが存在するか
   - 有効期限内か
   - この購入者が未使用か
3. カート内にクーポン対象の商品があるか確認
   - 対象商品がカートになければ適用不可
4. 対象商品の金額に対して割引を適用
   - 固定額の場合: 割引額 = min(クーポンの割引額, 対象商品の金額)
   - 割合の場合: 割引額 = 対象商品の金額 × 割引率
5. 注文確定 → クーポンを使用済みに記録

## 注文キャンセル時

- 注文がキャンセルされたらクーポンの使用記録を削除し、再利用可能にする

## 将来の拡張候補（今回は対象外）

- 管理者（admin）によるサイト全体クーポン発行
- 複数クーポン併用（出品者が異なる場合）
- カート画面での割引プレビュー
- 最低注文金額の条件
- 全体の利用回数上限
